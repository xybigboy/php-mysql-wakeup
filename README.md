# php-mysql-wakeup 远程开机项目只支持触摸设备
使用环境： mysql +php5.6
支持设备：带触摸的设备都支持
功能 ：发送开机命令，远程开机(需要nps，内网穿透)
数据修改：手动加设置mac 其他不重要

步骤：导数据wakeup.sql
          修改mysql地址 cnn.php
访问即可
